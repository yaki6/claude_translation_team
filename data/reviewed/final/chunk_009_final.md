1. 从五个领域中识别决定性的影响因素，并简要描述。
2. 推导这些影响因素对贵公司产生具体后果的不同层级（时间、强度）。
3. 根据分级的具体后果绘制贵公司的脆弱性曲线：

脆弱 = 易碎
强健/有韧性 = 暂时抵抗或在低强度下抵抗
反脆弱 = 学习、自我强化
4. 决定要消除哪些脆弱性，以及如何消除。

# 照镜子

「许多人对他人的弱点有敏锐的眼光，对自己的弱点却视而不见。」

恩斯特·费斯特尔，奥地利教师和作家

持续质疑自己和公司的能力，是最重要的个人领导力素质。经验知识和过往成功常常让高管在某些领域产生盲点。自我意识成为进步的障碍。不愿谈论自身缺陷是人之常情。此外，权力和领导地位带来的孤立感，往往导致缺乏坦诚的反馈。虽然反馈经常被索求，却很少被真正要求或接受。如果你的团队中还没有一个蒂尔·尤伦斯皮格尔（中世纪小册子中的主角，通过恶作剧揭露同时代人的恶习）这样的角色，那么是时候让某人为你照镜子了。

在分析了公司的外部环境之后，现在到了关键时刻。你的业务将被毫不留情地置于显微镜下审视，因为战略的潜在行动空间只有在环境与自身能力相结合时才会显现。但应该从哪里开始？通常，某些方面确实存在问题，但并非一切都糟糕透顶。

蒂尔·尤伦斯皮格尔用恶作剧装傻时，他狡猾地假装愚钝，按字面意思理解每一句话。通过这种方式，他为同胞们照镜子，展示他们的弱点和过失。通过笑声和幸灾乐祸，他公开地让他们接受尖锐但未言明的批评，揭示那个时代的缺陷。你不必如此毫不留情地展示所有洞察，但管理团队内部的分析确实需要残酷的诚实。

有两个关键问题能帮助揭示缺陷和低效：你今天真正的盈利来源是什么？如果有三个愿望，你明天会立即改变什么？任何事都可以！这听起来像恶作剧吗？那就立即着手解决这些问题，诚实地回答——无论是独自思考、与团队讨论，还是与外部人士交流。

然而，公司最终从哪些产品和服务中获利最多，这个问题的答案往往长期保持神秘。是的，你的组织架构、控制和报告系统往往过于复杂。但要相信你自己和团队。你们会找到答案——那些产品

# 自身现实

除了分析关键绩效指标和盈利来源，你还必须仔细观察和倾听。当然，这对身处高位的管理层来说并非易事。因此，正如在定性探索性访谈章节中所建议的，你应该考虑是否需要引入外部的新视角。与关键内部人员的几次会谈可能成为产生关键差异的魔法成分。重要的是为所有参与者提供一个熟悉的安全空间——同事或员工表达的真相不能被事后曲解或负面归咎。这里有一份关键问题清单，帮助你彻底审视公司，直面真相。

## 关键问题：

1. 我们关键绩效指标的五年趋势如何，能从中得出什么结论？
2. 我们在哪里赚钱，在哪里亏损？
3. 如何处理那些不盈利的产品？
（问题2到3需要按客户、产品组和地理区域分解利润）
4. 我们有哪些可以转化为竞争优势的主要优势？
5. 哪些弱点阻碍了我们取得更好的业绩？
6. 如果有三个愿望，我们会立即改变什么？

## 分析I：利润集中图

在右栏中，按降序列出最盈利的产品或产品类别，标明它们对总利润的贡献比例。当识别出80%的总利润时画一条线。在这条线下方列出构成剩余20%利润的产品或产品类别。如果清单过长，应当精简，以便专注于核心利润创造者。

## 提示

不能使用平均数字。平均数字毫无意义——相反，它们掩盖真相！提升业绩的方法是改善或剔除表现不佳的部分。这样平均值自然上升。因此，分解关键绩效指标至关重要——只有这样才能了解哪里盈利良好，哪里存在问题。

## 分析II：优势和劣势

## 选项A：探索性访谈

在优势/劣势矩阵中列出从访谈中归纳的洞察。用同事的精辟匿名引语来说明某些观点。

## 选项B：头脑写作法

如果没有内部资源进行探索性调查，可以与管理团队进行头脑写作会议。每位参与者拿到一张空白纸，列出感知到的优势和劣势。几分钟后，每人将纸传给邻座。接收者只能在前人基础上补充优势或劣势，不能删除。重复这个过程，直到纸张回到原主人手中。然后向全组展示洞察，对陈述的优势和劣势进行评估。最后将结果填入提供的表格。

# 不做半途而废的事

「追逐两只兔子的人一只也抓不到。」
孔子

现状分析中每个模块的所有数据都已充分收集、分析和可视化。你已经获得了一些初步洞察。接下来是工作流程的下一步：聚焦。

你现在必须获得更深层的洞察并做出决策，为公司确立明确的焦点。承担所有后果，没有如果或但是。商业战略大师迈克尔·波特（1996）总结道：
「战略的本质是选择不做什么。」

决定不做什么对人类来说往往很困难，因为我们不喜欢提前限制自己，出于规避风险的考虑，我们更愿意为自己保留更多而非更少的回旋余地。但在资源有限的情况下，如果想做那些让我们「独特」并追求卓越的事情，就必须定义「不做的事」。因此，既非「这个或那个」，也非「这个和那个」，而是「这个，但不是那个」。因为那个不属于我们的DNA，或与我们的自我认知相矛盾。我们不能成为「万事通」。我们必须选择。

Netflix纪录片系列《寿司之梦》有力地证明了这一点。纪录片讲述了85岁寿司大师小野二郎的故事。在某个阶段，他决定在东京地铁站经营一家只有10个座位的寿司店。我们见证了他如何通过自我牺牲般的专注和对专业领域精通的追求，在极小的规模上创造了真正伟大的成就：每盘300美元，米其林三星，美食家心中的传奇。正如日本作家谷崎润一郎所说：「精通在于……多年不懈打磨形成的包浆。」也许包浆的意象能成为聚焦公司业务、达到精通境界的灵感或主题。

## 从现状分析到目标愿景

虽然到目前为止你只是间歇性地让个别同事参与规划、收集和分析，但现在应该邀请整个管理团队参加第一次集体战略研讨会，必要时可扩大到包括选定的发起人。

---


第一项任务是回答：「我们公司正在发生什么，我们今天处于什么位置？」从现状分析结果中创建管理团队对公司现状的共识，并据此制定未来战略的目标愿景，这是基本的管理职责。花时间作为团队详细讨论现状分析的结果。鼓励参与者质疑即使是明显的关联，寻找异常值和反常现象。提问总是有帮助的。就像哥伦比亚商学院管理实践教授、前利华食品和特洛皮卡纳CEO威利·彼得森在热身时向听众提出的问题：

- 你的产品或服务是否有「制胜主张」，即创造比竞争对手更多价值的价值主张？你比对手做得更好或不同的是什么？
- 你在哪里输给他们？你的公司在哪里不如竞争对手？
- 你的哪些产品或服务是盈利的？
- 你的制胜主张也是盈利主张吗？
- 你在哪里获得比竞争对手更高的利润？

讨论这些问题及更深层问题的答案，将使「这里发生了什么」的总体答案明晰化，即事实和真相。具体而言：结果的原因是什么，背景是什么？例如，考虑你的产品组合：它如此庞大是因为想满足每个客户需求吗？还是在服务随机发展、未经仔细审视的客户细分？在当前竞争领域的挑战和机遇、潜在的新竞争领域、潜在的行动方案和相关风险以及可能的时间框架上，获得清晰认识和共识。

接下来，你必须确定焦点——公司未来应该在哪些竞争领域运营——并借助目标和关键成果来定义公司资源分配的优先级。这个过程的结果显示你需要做出哪些战略决策（如投资），以及必须采取哪些运营或战术措施来实现聚焦。无论初步得出什么有意义的战略：确保它与你对公司希望在世界上产生的影响的理念保持一致。这个全局评估将产生你的目标愿景：「我们要去哪里？」

# 战略研讨会 I

「那次在杜塞尔多夫与StrategyFrame®以及安德伍德先生和魏根德教授举行的战略研讨会，对我们公司来说是真正的转折点。」汉斯-于尔根·冯·格拉塞纳普，BILSTER BERG驱动度假村总经理

让StrategyFrame®工作流程中的第一次战略研讨会成为你和团队的特别时刻。让参与者即使在三年后仍然愿意积极回忆的经历。为深入面对你的环境和公司现实创造最佳条件。与团队就公司的起点进行深入讨论，并以此为基础制定未来方向的新想法。肯定会有一些意见分歧。这是过程的一部分，只要不涉及个人恩怨。通过这个研讨会，你也在为新的战略惯例奠定基础。

准备研讨会时，应明确以下要点：

```
#1: 目标
#2: 时长和节奏
#3: 参与者
#4: 会议地点
#5: 主持
```


## #1: 战略研讨会的目标是什么？

研讨会结束时，你和团队应该基于现状分析制定出共同的未来目标愿景。目标不是详细阐述整个战略。相反，如果想一次实现太多，你会失望。通常，并非所有参与者从一开始就具有相同的知识水平，因为他们可能只参与了现状分析的个别要素。因此，你必须让参与者「跟上节奏」，以便：

- 实现对初始状况的共同评估，
- 理解挑战和机遇，
- 开始就战略选项达成一致。

---


一天时间不够。我们一再遇到不愿「牺牲」超过一个工作日的领导者。时间是有限资源，但不要做错误的节约。通常，在所有参与者在精神上和智力上到位并达成对初始状况的认可评估之前，已经过去了不止一天。

理想情况下，应该为现状分析后的第一次研讨会计划2.5天。半天预留给到达会议地点后的下午。可以用来介绍研讨会议程并展示现状分析的核心结果。与所有参与者共进晚餐结束这一天。这样第二天早上可以直接开始，而不必等待仍在路上的同事。我们估计需要接下来的一天半来达成对初始状况的共同理解，评估风险和机遇，并制定潜在解决方案。剩余的半天用于起草目标愿景和规划下一步。

除了任务之外，是否需要额外的团队活动，或者邀请攀登过最高峰或完成铁人挑战的励志演讲者？在这里，少即是多。你的目标明确。研讨会应该为后续流程提供具体成果。因此我们不建议除此之外的任何活动。我们也不建议邀请对你的战略发展没有直接贡献或在战略过程中不会发挥重要作用的嘉宾演讲者。

# #3: 谁应该参加战略研讨会？

不言而喻，高级管理团队应该参加研讨会，因为这些高管将负责在公司所有领域实施制定的战略。因此，早期参与至关重要。当然，你可以扩大范围，纳入其他人以获得额外的视角和灵感。

但请记住：从10到12人以上，沟通和协调的复杂性会显著增加。你需要加大激励力度，防止个别参与者退缩、关闭或脱离。此外，每个不属于管理团队的参与者都会改变团队动态——这可能产生积极或消极的影响。在这种情况下，你应该考虑额外的团队建设措施，以确保研讨会的接受度和凝聚力。
